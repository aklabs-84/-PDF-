# 마크다운 노트 (Markdown Note) - 작업 로그 및 해결사례 모음

## [2026-02-25] 학습 가이드 튜토리얼 기능 및 아크랩스 링크 추가

### 1. 주요 변경 사항 및 추가 기능
*   **Aklabs 홈페이지 링크 추가:** 앱 저장소의 메인 문서인 `README.md` 하단 기능 설명부 아래에 `Aklabs (아크랩스)` 관련 링크 (https://litt.ly/aklabs) 영역을 신규 할당.
*   **튜토리얼 / 사용 가이드 추가:** 앱 설정 표시줄 ('내 문서', '도움말' 헤더 버튼 영역) 에 파란색으로 강조된 **'📖 학습 가이드'** 버튼을 새롭게 추가하여 사용성을 개선함. 
*   **가이드 렌더링 시스템:** 
    *   버튼 클릭 시 `ui-manager.js`의 `showGuide()` 메서드가 실행되어 에디터에 마크다운 튜토리얼 전문이 자동으로 로드되고 즉시 우측 미리보기 창에 반영되도록 조치.
    *   **데이터 유실 방지**: 사용자가 기존 문서를 작성하고 있는 도중(빈 문서가 아니거나 50자 이상일 경우)이라면, 작성 중이던 텍스트가 덮어씌워진다는 경고 모달창 (`confirm`)을 띄워 사용자 피드백을 한 차례 거치도록 설계.
*   **튜토리얼 문서 내용 (가이드):** 
    *   기본 서식(굵게, 기울임, 취소선), 스마트 체크리스트 도입.
    *   복잡한 `KaTeX` 기반의 수학 수식(적분 등) 및 `Mermaid` 기반 다이어그램 생성 예시 노출.
    *   코드 하이라이팅이 적용된 `JavaScript` 코드 렌더 블록, 3단 표(Table) 예제 등 앱 내에서 수행 가능한 모든 슬래시(`/`) 커맨드 핵심 기능을 실제 작동 예시와 함께 한눈에 파악할 수 있도록 구성.

### 2. 해결한 부분 및 작성 팁
*   버튼 UI 작업 시 기존의 `Tailwind CSS` 버튼들과 시각적인 일관성을 유지하면서도 기능의 특성에 맞게 글꼴 옵션(`font-bold`, `text-blue-600` 등)과 시각적 요소(파란색 윤곽선)를 조정하여 '가이드'라는 목적에 맞게 버튼 가시성을 높였습니다.
*   `ui-manager.js` 내에 아주 긴 튜토리얼 템플릿(가이드 마크다운 소스)을 백틱(\`\`)을 활용해 렌더 구조 변경 없이 한 덩어리로 잘 주입했으며, 내용 변경과 동시에 우측 하단의 `Toast` 알림창 팝업을 활용해 시스템 상태를 사용자에게 명확히 피드백 하도록 UX를 매끄럽게 구성했습니다.
*   **Mermaid 9.4.3 문법 오류 해결**: 가이드 예제 코드 중 `E[슬래시(/) 치기]` 부분에서 노드 텍스트 안에 있는 특수 문자 괄호와 슬래시 때문에 렌더링 Syntax 에러가 발생하는 것을 발견했습니다. 이를 방지하기 위해 특수문자를 제거한 `E[명령어 입력하기]` 로 텍스트를 순화하여 호환성을 확보했습니다.
*   **Mermaid 렌더링 누락 버그 해결**: 마크다운에서 `mermaid` 코드 블록이 `highlight.js`와 겹쳐서 일반 코드로 인식되어 다이어그램이 렌더링되지 않던 사이드 이펙트를 해결했습니다. `marked` 설정과 `mermaid.init` 호출 로직을 수정하여 원래 코드를 `data-mermaid-src`에 보존하고, 실시간 반응형으로 업데이트될 때마다 기존 SVG를 지우고 다시 올바르게 렌더링하도록 `editor-manager.js`를 수정했습니다.
